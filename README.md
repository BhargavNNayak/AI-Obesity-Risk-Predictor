# Obesity Risk Level Classifier

## Overview

Machine Learning based obesity risk prediction system using lifestyle and health data.

## Features

- Multi-class obesity prediction
- Random Forest & Gradient Boosting
- Streamlit Web Application
- Flask REST API
- BMI Calculation
- Health Recommendations

## Tech Stack

- Python
- Scikit-learn
- Streamlit
- Flask
- Pandas
- NumPy

## Run Project

### Install Requirements

pip install -r requirements.txt

### Train Model

python train_model.py

### Run Streamlit App

cd app

streamlit run streamlit_app.py

### Run Flask API

python flask_api.py
